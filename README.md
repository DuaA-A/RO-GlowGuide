# RO — Skincare and Haircare Guide

A comprehensive digital catalogue and clinical guide for personalized skincare and haircare regimens. This platform provides structured routines for various skin and hair conditions based on professional care standards.

## Project Overview

This project serves as a centralized resource for users to understand their skin and hair types and find appropriate treatment paths. It includes a curated product catalogue with detailed ingredient analysis and medical safety information.

## Core Features

### Skincare Analysis
- Detailed guides for Acne, Rosacea, Eczema, and Hyperpigmentation routines.
- Classification systems for Oily, Dry, Combination, and Sensitive skin types.
- Product database with verified ingredient lists.

### Haircare Solutions
- Specialized protocols for Hair Loss, Scalp Eczema, and Split Ends.
- Texture-based guidance for Straight, Wavy, Curly, and Coily hair.
- Safety information including pregnancy contraindications for active treatments.

### User Interface
- Modern, responsive design optimized for high-readability.
- Animated transitions for a seamless browsing experience.
- Advanced search and filtering system for the product catalogue.

## Technical Stack

- Frontend: React 18
- Build Tool: Vite
- Language: TypeScript
- Styling: Tailwind CSS
- Animations: Framer Motion
- UI Icons: Lucide React

## Getting Started

### Installation

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install the required dependencies:
   ```bash
   npm install
   ```

### Development

To start the development server and view the project locally:
```bash
npm run dev
```

### Production Build

To create an optimized production build:
```bash
npm run build
```

## Project Structure

- `src/app/components`: Reusable UI elements and layout modules.
- `src/app/data`: Clinical data files for skincare and haircare products and routines.
- `src/app/pages`: Main application views and page structures.
- `src/app/routes`: Configuration for application navigation.
- `src/styles`: Global styles and theme definitions.
